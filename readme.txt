ソフト名：　NeoMupl
バージョン：0.33.0
種別：　　　フリーウェア
内容：　　　音楽を再生(ループ対応)
動作環境：　.NETが動く環境
作者：　　　美文

●概要
音楽ファイルを再生します。
自己満足だけのソフトでいろいろ足りてませんのでご注意ください。

●動作環境
.NET 4.6以降が動く環境
以下の環境で動作確認済みです。
Windows 10 Home 64bit

●アーカイブ内容
アーカイブには以下のファイルが含まれています。
NeoMupl.exe       ：実行ファイル
NeoMupl.exe.config：実行ファイルの設定
*.dll             ：音声再生用DLL
readme.txt        ：このテキスト
license.txt       ：ライセンス

●あとから作られるファイル
NeoMupl.ini：設定ファイル
*.nmp      ：リストデータ

●使い方
ソフト本体の操作方法は言わなくても大体わかると思います。

●注意事項
・本ソフトウェアはフリーソフトウェアです。
・本ソフトウェアの利用に際して料金がかかることはありません。
・著作権は美文が保有します。
・本ソフトウェアによるトラブルに関して作者は責任を負いません。
・複数ユーザーによる同時利用を想定していません。
・現時点でDirectMusicの時間表示が変です。
・現時点でMCIのループ位置指定は無効です。
・現時点でNAudioの再生速度指定は無効です。

●よくあるかもしれない質問
きっと多く来るだろう質問を予測して先手を打って答えておきます。

・MIDIを再生するとフリーズします/突然落ちます
ポートとの相性のようです。
フリーズしないポートにするか、DirectShowを使ってください。

・起動時に例外が出ます
その場で終了してからもう一度起動すると直ることがあるようです。
データのバックアップは取っておいたほうがいいと思います。

・MCIでMIDIが再生できません
DirectMusicと競合して負けてるようです。
MCI実装したのはいいけどいいところなしだなぁ…。
