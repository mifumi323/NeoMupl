ソフト名：　NeoMupl
バージョン：0.30
種別：　　　フリーウェア
内容：　　　音楽を再生(ループ対応)
動作環境：　.NETが動く環境
作者：　　　美文

●概要
音楽ファイルを再生します。
自己満足だけのソフトでいろいろ足りてませんのでご注意ください。

●動作環境
.NETが動く環境
以下の環境で動作確認済みです。
Windows 10 Home 64bit

●アーカイブ内容
アーカイブには以下のファイルが含まれています。
NeoMupl.exe       ：実行ファイル
NeoMupl.exe.config：実行ファイルの設定
*.dll             ：音声再生用DLL
readme.txt        ：このテキスト
license.txt       ：ライセンス

●あとから作られるファイル
NeoMupl.ini：設定ファイル
*.nmp      ：リストデータ

●使い方
ソフト本体の操作方法は言わなくても大体わかると思います。

●よくあるかもしれない質問
きっと多く来るだろう質問を予測して先手を打って答えておきます。

・MIDIを再生するとフリーズします/突然落ちます
ポートとの相性のようです。
フリーズしないポートにするか、DirectShowを使ってください。

・起動時に例外が出ます
その場で終了してからもう一度起動すると直ることがあるようです。
データのバックアップは取っておいたほうがいいと思います。

・MCIでMIDIが再生できません
DirectMusicと競合して負けてるようです。
MCI実装したのはいいけどいいところなしだなぁ…。

・しばらく動かしてたら勝手にエラーで落ちます
ごめんなさい。原因不明です。直せるものなら直したいです。
エラーで落ちる前に再起動すればエラーで落ちませんが、
たぶんそんな回答は誰も望んでないと思います。
