ソフト名：　NeoMupl
バージョン：0.29
種別：　　　フリーウェア
内容：　　　音楽を再生(ループ対応)
動作環境：　.NETが動く環境
作者：　　　美文

●概要
音楽ファイルを再生します。
まだ開発途上の不完全なソフトでいろいろ足りてませんのでご注意ください。

●動作環境
.NETが動く環境
以下の環境で動作確認済みです。
WindowsVista Home Premium SP1

●アーカイブ内容
neomupl.lzhには以下のファイルが含まれています。
DM7lib.dll               ：DirectMusic
Interop.QuartzTypeLib.dll：DirectShow
NeoMupl.exe              ：実行ファイル
NeoMupl.txt              ：このテキスト

●ファイル構成
DM7lib.dll               ：DirectMusic
Interop.QuartzTypeLib.dll：DirectShow
NeoMupl.exe              ：実行ファイル
NeoMupl.txt              ：このテキスト
NeoMupl.ini              ：設定ファイル。自動生成されます。
*.nmp                    ：リストデータ

●使い方
ソフト本体の操作方法は言わなくても大体わかると思います。

●よくあるかもしれない質問
きっと多く来るだろう質問を予測して先手を打って答えておきます。

・.NETがなくて起動できません！
「Windows Update」で「Microsoft .NET Framework 2.0」を
インストールしましょう。それでも無理ならもう知りません。

・MIDIを再生するとフリーズします/突然落ちます
ポートとの相性のようです。
フリーズしないポートにするか、DirectShowを使ってください。

・起動時に例外が出ます
その場で終了してからもう一度起動すると直ることがあるようです。
データのバックアップは取っておいたほうがいいと思います。

・MCIでMIDIが再生できません
DirectMusicと競合して負けてるようです。
MCI実装したのはいいけどいいところなしだなぁ…。
